## ğŸ“– How to Use

### View the Website
1. Clone or download this repository
2. Open `index.html` in your browser
3. Enjoy the portfolio!

### Test Responsiveness
1. Open `index.html` in Firefox or Chrome
2. Press `F12` to open DevTools
3. Press `Ctrl + Shift + M` to toggle responsive design mode
4. Select different devices to test:
   - iPad Pro 12.9" (tablet)
   - iPhone 12 Mini (mobile)
   - Or use custom widths

### Customize for Yourself
- Replace `images/profile.png` with your own picture
- Update your name, bio, and projects in HTML
- Modify color scheme in `:root` CSS variables
- Add more skills and projects as needed
